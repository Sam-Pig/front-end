背景

2077年10月23日核战爆发。主角一家进入了附近一家避难所科技运营的111避难所。最后所有人被进行了低温休眠，主角因此失去了意识。 过了不久，避难所的冷冻仓被关闭，主角眼睁睁地看着一个雇佣兵杀了自己的配偶，夺走了尚在襁褓中的孩子，并再次启动低温休眠。 后来冷冻仓发生了“故障”，主角得以离开111号避难所，进入联邦废土，试图寻找孩子的踪迹，此时距离2077年已经过去了210年。

队友

限制一人。提升他们的好感及完成他们的任务后，可以获得他们的一项能力。与部分队友对话可以得到简单的物品（有冷却）。 也可通过mod带多人，但鉴于他们的爱好相互冲突，需要格外注意。 警告：与他们出身的集团敌对会让他们立即攻击你。 总是做队友厌恶的事情会导致他们反目。好感到达临界点的时候会触发一次警告对话，继续做则会触发第二次警告对话，此时可选择说服或决裂（壮壮无法说服）。说服后再做他们则会直接攻击玩家。决裂的队友无法再次招募。 队友不会死亡，只会瘫痪。瘫痪时使用治疗针可以恢复。战斗结束后瘫痪的队友会复原。在安装了“机械守卫”dlc后（达到15级接任务解锁）机器人有了自己的专属治疗针。 决裂的队友可以被玩家杀死（普雷斯顿和居里除外）。

S.P.E.C.I.A.L系统

每升级一次可增加一点special或perk。

  • Strength力量
    力量是你体力的衡量标准。 它影响你可以携带多少，并确定所有近战攻击的有效性。
  • Perception感知
    感知是你对周围环境的“第六感”，并影响武器准确度。
  • Endurance耐力
    耐力是整体身体素质的衡量标准。 它会影响您的健康状况以及冲刺中的行动点流失。
  • Charisma魅力
    魅力是你的魅力和说服别人的能力。 它影响你在对话中说服他人和讨价还价的能力。
  • Intelligence智力
    智力是衡量您的整体精神敏感度，并影响获得的经验积分数。
  • Agility敏捷
    敏捷度是衡量您整体的精细度和反应能力。 它影响了V.A.T.S.中的行动点的数量和你潜行的能力。
  • Luck幸运
    运气是衡量您的一般幸运，并影响重击出现率。当幸运为9时，暴击会有几率出现幸运草(四叶草)。
